## middlewareの作成

まぁ基本的にはドキュメントの通りにやっていく
https://react-router-docs-ja.techtalk.jp/how-to/middleware#1-context-%E3%82%92%E4%BD%9C%E6%88%90%E3%81%99%E3%82%8B


導入したいルートにミドルウェアを定義する
今回の場合、layoutとかがいいかもね

v8_middleware: true にすると、React Router 側の仕様が
「context は RouterContextProvider インスタンスであること」
にしないといけないので修正を入れる

じゃなくてルートで、ミドルウェアを有効にしたいときRouterContextProviderにしないといけなということだった

この時、DBヘルパーとか自前で定義しているならここの引数の型もいるので注意

ドキュメントと違うのは、userの定義とかを雑にインファイルでやっていること

フラグの有効か
react-router.config.ts
```ts
import type { Config } from "@react-router/dev/config";

export default {
  future: {
    v8_middleware: true,
  },
} satisfies Config;
```

app/context.tsの作成

 ```ts
 import { createContext } from "react-router"

export interface User {
  id: number
  name: string
  email: string
}

export const userContext = createContext<User | null>(null)

 ```

app.tsで、AppLoadContextをRouterContextProviderに修正する
claudeflareの設定があるので不要な人は不要

```ts
import { createRequestHandler, RouterContextProvider } from "react-router";

declare module "react-router" {
	export interface RouterContextProvider {
		cloudflare: {
			env: Env;
			ctx: ExecutionContext;
		};
	}
}

const requestHandler = createRequestHandler(
	() => import("virtual:react-router/server-build"),
	import.meta.env.MODE,
);

export default {
	async fetch(request, env, ctx) {
		const context = new RouterContextProvider();
		Object.assign(context, {
			cloudflare: { env, ctx },
		});
		return requestHandler(request, context);
	},
} satisfies ExportedHandler<Env>;

```

## supabaseでの認証を実装していく

やりたいこと
- 認証ないとログイン画面へ遷移させる
- ログイン中ならユーザ情報を表示する

supabaseはローカルでたてて、ユーザは作成済みです。

参考
https://supabase.com/docs/guides/auth/server-side/migrating-to-ssr-from-auth-helpers
https://supabase.com/docs/guides/auth/server-side/creating-a-client?queryGroups=framework&framework=react-router&queryGroups=environment&environment=react-router-loader

ログイン用の処理の関数
supabaseのクライアントを作成する関数

一連の流れをやってみる

```ts

function createSupabaseClient(request: Request, headers: Headers) {
	return createServerClient(
		"http://127.0.0.1:54321",
		"your-supabase-anon-key",
		{
			cookies: {
				getAll() {
					return parseCookieHeader(request.headers.get("Cookie") ?? "").filter(
						(cookie): cookie is { name: string; value: string } =>
							cookie.value !== undefined,
					);
				},
				setAll(cookiesToSet) {
					cookiesToSet.forEach(({ name, value, options }) => {
						headers.append(
							"Set-Cookie",
							serializeCookieHeader(name, value, options),
						);
					});
				},
			},
		},
	);
}
const authMiddleware: Route.MiddlewareFunction = async (
	{ request, context },
	next,
) => {
	const headers = new Headers();
	const supabase = createSupabaseClient(request, headers);

	const { data } = await supabase.auth.getUser();

	if (data.user) {
		console.log("ログイン済みです");

		context.set(userContext, {
			id: data.user.id,
			name:
				typeof data.user.user_metadata?.name === "string"
					? data.user.user_metadata.name
					: (data.user.email ?? "テストユーザー"),
			email: data.user.email ?? "test-user@example.com",
		});
		const response = await next()
    const responseWithHeaders = new Response(response.body, {
      status: response.status,
      statusText: response.statusText,
      headers: response.headers,
    })
    for (const [key, value] of headers.entries()) {
      responseWithHeaders.headers.append(key, value)
    }
    return responseWithHeaders
	}

	console.log("未ログインです");
};

export const middleware: Route.MiddlewareFunction[] = [authMiddleware];

export async function loader({ context, request }: Route.LoaderArgs) {
	const existingUser = context.get(userContext);
	if (existingUser) {
		console.log("既にログイン済みです");
		return { user: existingUser };
	}

	// ログインしていない場合、ログインを試行
	console.log("ログインをします");

	const headers = new Headers();
	const supabase = createSupabaseClient(request, headers);
	const { data: signInData, error: signInError } = await supabase.auth.signInWithPassword({
		email: "wawawa@example.com",
		password: "password",
	})
	console.log("[home loader] signIn data", signInData)
	console.log("[home loader] signIn error", signInError)

	const { data: userData, error: userError } = await supabase.auth.getUser();
	console.log("[home loader] getUser after login data", userData);
	console.log("[home loader] getUser after login error", userError);

	const loggedInUser = userData.user
		? {
				id: userData.user.id,
				name:
					typeof userData.user.user_metadata?.name === "string"
						? userData.user.user_metadata.name
						: (userData.user.email ?? "テストユーザー"),
				email: userData.user.email ?? "test-user@example.com",
			}
		: null;
	if (loggedInUser) {
		context.set(userContext, loggedInUser);
	}

	return data({ user: loggedInUser }, { headers });
}

export default function Home({ loaderData }: Route.ComponentProps) {
	console.log("[home component] user", loaderData.user);

```
